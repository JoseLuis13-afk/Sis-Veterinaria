# 🐾 Sistema de Gestión de Veterinaria

Un sistema web completo y escalable para la administración integral de clínicas veterinarias, desarrollado con el framework **Laravel 11**. Este proyecto permite gestionar desde el historial médico de las mascotas y la programación de citas, hasta el control de inventario, flujo de caja y roles del personal.

## ✨ Características Principales

* **Control de Acceso y Roles:** Sistema de seguridad y permisos para asegurar que cada usuario (ej. Administradores, Veterinarios, Recepcionistas) tenga acceso únicamente a los módulos correspondientes.
* **Gestión Médica y de Clientes:**
  * CRUD completo de Clientes y Usuarios.
  * Registro y gestión de Mascotas (historiales, imágenes).
  * Sistema de Citas interactivo mediante calendario.
* **Control Comercial y Financiero:**
  * Módulo de Productos y Categorías.
  * Control de Inventario en tiempo real.
  * Registro de Compras y Ventas.
  * Apertura, control y cierre de Cajas.

## 🛠️ Stack Tecnológico y Plugins

Este proyecto está construido con herramientas modernas para asegurar una experiencia de usuario rápida y dinámica:

* **Backend:** [Laravel 11](https://laravel.com/)
* **Panel de Administración:** Plantilla [AdminLTE](https://adminlte.io/)
* **Tablas Dinámicas:** DataTables (para búsquedas y paginación eficiente)
* **Notificaciones:** SweetAlert2 (alertas amigables e interactivas)
* **Formularios:** Select2 (selectores con buscador integrado) y CKEditor (editor de texto enriquecido)
* **Calendario:** FullCalendar (gestión visual de las citas médicas)
